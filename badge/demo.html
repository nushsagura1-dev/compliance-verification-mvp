Compliance Badge Demo

Add data-domain="your-domain.com" to any element and include the badge script. The badge fetches live status from the API and renders automatically.

example.com
status: active
revoked.com

<div id="compliance-badge" data-domain="example.com"></div>
<script src="https://your-api.com/badge/badge.js"></script>
